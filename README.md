# Расширение "Fetch Tracker"

Это расширение для браузера, которое помогает отслеживать сетевые запросы, выполняемые веб-страницами. Оно предоставляет следующие функции:

- **Мониторинг Fetch-запросов**: Отслеживайте все вызовы `fetch` API, выполняемые активной вкладкой.
- **Копирование деталей запроса**: Легко копируйте URL fetch-запроса в буфер обмена.
- **Запуск/Пауза отслеживания**: Контролируйте, когда начинать и останавливать мониторинг fetch-запросов.
- **Просмотр истории запросов**: Просматривайте список всех перехваченных fetch-запросов.

Это расширение полезно для разработчиков и всех, кому необходимо проверять сетевую активность в браузере.

## Как использовать

Для загрузки и использования этого расширения в браузере (например, Chrome или Edge):

1.  **Скачайте (со страницы релизов) или клонируйте репозиторий**: Получите все файлы проекта на свой компьютер.
2. **Распакуйте расширение**: Извлеките файлы архива в любую папку на вашем компьютере.
3.  **Откройте страницу управления расширениями**:
    *   В Google Chrome: Введите `chrome://extensions` в адресной строке и нажмите Enter.
    *   В Microsoft Edge: Введите `edge://extensions` в адресной строке и нажмите Enter.
4.  **Включите режим разработчика**: Найдите переключатель "Режим разработчика" (Developer mode) в правом верхнем углу страницы и включите его.
5.  **Загрузите распакованное расширение**: Нажмите кнопку "Загрузить распакованное расширение" (Load unpacked).
6.  **Выберите папку расширения**: В появившемся диалоговом окне выберите папку, в которой находятся файлы этого расширения (корневая папка проекта).
7.  **Закрепите расширение (по желанию)**: После загрузки расширение появится в списке. Вы можете закрепить его на панели инструментов браузера для быстрого доступа.

Теперь расширение готово к использованию! Нажмите на иконку расширения, чтобы открыть его панель управления.
