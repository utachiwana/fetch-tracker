# Расширение "Fetch Tracker"

Это расширение для браузера, которое помогает отслеживать сетевые запросы, выполняемые веб-страницами. Оно предоставляет следующие функции:

- **Мониторинг Fetch-запросов**: Отслеживайте все вызовы `fetch` API, выполняемые активной вкладкой.
- **Копирование деталей запроса**: Легко копируйте URL fetch-запроса в буфер обмена.
- **Запуск/Пауза отслеживания**: Контролируйте, когда начинать и останавливать мониторинг fetch-запросов.
- **Просмотр истории запросов**: Просматривайте список всех перехваченных fetch-запросов.

Это расширение полезно для разработчиков и всех, кому необходимо проверять сетевую активность в браузере.
